# Todo List App

Aplicación de lista de tareas utilizando Vue, Vite y Tailwindcss

## Instalación

```
$ git clone https://github.com/faviofz/todolist-vue-vite-tailwindcss.git
$ cd todolist-vue-vite-tailwindcss
$ npm install
$ npm run dev
```
